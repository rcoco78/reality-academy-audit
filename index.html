Roadmap

Plan 4–6 semaines

Roadmap que je prends en charge et que nous pouvons exécuter ensemble.

S1 — États + stop‑rules + formulaires
  • Valider schéma d’états & événements dans `Sellsy`.
  • Activer stop‑rules Make (SQL ➔ stop nurture; annulé ➔ décrément score; no‑show ➔ relance).
  • Durcir formulaires (email pro, phone regex, reCAPTCHA, double opt‑in Mailjet).
  • Définir la base maîtresse (contacts/entreprises) + règles strictes de déduplication.
S2 — Domaine secondaire + personas + AB test
  • Créer domaine secondaire outbound + warm‑up; 2–4 inbox/persona.
  • Lancer AB test « Calendrier RH » (budget suffisant/variante).
  • Démarrer collecte continue + enrichissement; QA hebdo « data health ».
S3 — Dashboard funnel + SEO + contenus
  • Mettre en place dashboard hebdo (funnel complet).
  • Coordination partenaire SEO (correctifs Core Web Vitals) + brief 2–3 articles.
S4–S6 — Scale outbound + ads + salons
  • Industrialiser outbound (séquences persona, déduplication CRM, scoring).
  • Coordination itérations Ads via agence/équipe; playbook salons.
